What an OTDR is actually doing

An OTDR sends a pulse of light into the fiber and listens for backscatter and reflections coming back. That lets it estimate distance to events, event loss, reflectance, total loss, and the general attenuation slope of the fiber.

Think of the trace like a canyon echo map: flat-ish slope means normal fiber loss, a step down means loss, a spike means reflection, and a cliff at the end usually means you found the fiber end or a break.

Fast screen-reading order

  • Check wavelength being used
  • Find launch cable end before judging first connector
  • Look for spikes = reflective events
  • Look for steps = loss events
  • Compare 1310 and 1550 if a bend is suspected
  • Confirm whether the far end is real or a ghost

VIAVI Smart Event Icons

Front Connector / Test Port
The OTDR's own front-end connection.
Launch Cable
Reference fiber between the OTDR and the fiber under test. Helps get past the dead zone and measure the first connection.
Splice
Usually a non-reflective loss event. Fusion splices should be small and clean.
Connector
Reflective event. Usually shows a spike plus some loss.
Bend
Macrobend or stress point. Often worse at 1550 nm than at 1310 nm.
Bend on Connector
Loss behavior associated with a connector area plus bend characteristics.
Splitter / 2x Splitter / X Splitter
Large non-reflective loss event used in PON networks.
MUX / DMUX
WDM component causing insertion loss, often with low reflection if APC connected.
Fiber End
End of the tested fiber or an open end. Often a strong reflection if open.
Ghost
A false event caused by strong reflections bouncing around. Sneaky trace mirage.

Core Definitions

Event
Any measurable feature on the OTDR trace, such as a splice, connector, bend, splitter, break, or fiber end.
Attenuation
Normal signal loss along the fiber, usually shown as the slope of the trace in dB/km.
Event Loss
The dB drop caused by one event, like a splice or connector.
Reflectance
How much light is reflected back from one event. More negative dB values are better.
ORL
Optical Return Loss. Total reflected light coming back from the whole link. It is commonly shown as a positive number. Higher is better.
Dead Zone
The blinded section after a strong reflection or at the OTDR front end where nearby events cannot be separated clearly.
Launch Cable
Also called a pulse suppressor. Lets the OTDR recover from the launch pulse before the first real event.
Receive Cable
Fiber placed at the far end so the last connector can be measured properly.
Bidirectional Test
Testing from both ends and averaging results to improve accuracy, especially across dissimilar fibers or suspicious splice readings.
Gainer
An event that appears to have negative loss because of backscatter coefficient differences between two fibers. It can fool a one-way test.

Bend clues

  • If a bend is suspected, test at both 1310 nm and 1550 nm.
  • Average bend loss may be several times greater at 1550 nm.
  • A large bend or kink can look like the end of fiber if the loss is extreme.
Fusion splices and normal bends are generally expected to show little to no reflection.

Dead Zone Basics

  • Event dead zone: how close two events can be before the OTDR can no longer separate them.
  • Attenuation dead zone: how long it takes after a reflective event before accurate loss measurement can resume.
  • Shorter pulse widths improve resolution but reduce test range.

Launch / Receive Cable Rules

  • Always use a launch cable to measure the first connector properly.
  • Use a receive cable if you need to measure the last connector at the far end.
  • Keep these cables clean and known-good or they become part of the problem parade.

Cursor / Table Readout

  • Distance: where the event is
  • Loss: dB added by the event
  • Reflect: reflectance of the event
  • Slope: dB/km before the event
  • Total loss: accumulated link attenuation

Troubleshooting quick guide

Big spike

Usually a connector, open end, or break. Check whether it is real or a ghost created by a previous strong reflection.

Step down with no spike

Usually splice loss, splitter loss, or a non-reflective stress point.

Worse at 1550 nm

Strong bend suspect. Compare with 1310 nm before calling it a cut.

Negative loss or strange gain

Possible gainer effect from backscatter mismatch. Bidirectional testing can reveal the truth.

Remember this

  • Higher ORL is better
  • More negative reflectance is better
  • Short pulse = more detail
  • Long pulse = more distance
  • Dirty connectors create fake drama
